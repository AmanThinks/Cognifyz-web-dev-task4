# Cognifyz-web-dev-task
🚀 Cognifyz Web Developer Internship – Landing Page

📌 Project Overview

This project is a responsive internship landing page developed as part of Level 3 Task 2 during the Web Developer Internship at Cognifyz Technologies.

The goal of this project was to design a modern, responsive, and visually engaging landing page that presents internship information, highlights key features, and provides a smooth user experience across both desktop and mobile devices.

---

✨ Features

- Modern and Responsive Landing Page
- Animated Background UI
- Responsive Navigation Bar with Hamburger Menu
- Internship Highlights Section
- Qualifications Section
- Interactive Cards
- Call-to-Action (Apply Now)
- Smooth Hover and Scroll Animations
- Mobile Friendly Layout

---

🛠 Technologies Used

- HTML5 – Structure of the website
- CSS3 – Styling, layout, and animations
- JavaScript – Interactivity and navbar toggle
